# Крутая 3D Игра на Unity

## 🎮 Описание игры
Это полноценный 3D платформер с системой сбора монет, препятствиями и таймером.

## 📁 Структура проекта
```
Unity3DGame/
├── Assets/
│   ├── Scripts/
│   │   ├── PlayerController.cs      - Контроллер игрока
│   │   ├── CameraController.cs      - Камера от третьего лица
│   │   ├── CoinCollector.cs         - Сбор монет и очки
│   │   ├── RotatingCoin.cs          - Вращающиеся монеты
│   │   ├── MovingObstacle.cs        - Движущиеся препятствия
│   │   ├── FinishZone.cs            - Зона финиша
│   │   └── GameManager.cs           - Менеджер игры
│   ├── Prefabs/                     - Префабы объектов
│   ├── Scenes/                      - Сцены
│   └── Materials/                   - Материалы
```

## 🕹️ Управление
- **WASD / Стрелки** - Перемещение персонажа
- **Пробел** - Прыжок
- **Shift** - Бег (ускорение)
- **Мышь** - Вращение камеры (зажмите ЛКМ или ПКМ)
- **Колесо мыши** - Зум камеры
- **Escape** - Пауза

## ⚙️ Настройка в Unity

### Шаг 1: Создание сцены
1. Откройте Unity Hub и создайте новый 3D проект
2. Скопируйте папку `Assets` из этого репозитория в ваш проект

### Шаг 2: Настройка игрока
1. Создайте Capsule (GameObject → 3D Object → Capsule)
2. Добавьте компонент `Character Controller`
3. Добавьте скрипт `PlayerController`
4. Назначьте тег "Player"
5. Создайте слой "Ground" для земли

### Шаг 3: Настройка камеры
1. Поместите Main Camera как дочерний объект пустого GameObject
2. Добавьте скрипт `CameraController` на камеру
3. Укажите ссылку на игрока в поле Target

### Шаг 4: Создание уровня
1. Создайте Plane для земли (GameObject → 3D Object → Plane)
2. Назначьте земле слой "Ground"
3. Добавьте различные платформы
4. Разместите препятствия со скриптом `MovingObstacle`

### Шаг 5: Добавление монет
1. Создайте цилиндр или сферу для монеты
2. Добавьте скрипт `RotatingCoin`
3. Разместите монеты по уровню

### Шаг 6: Настройка менеджера игры
1. Создайте пустой GameObject "GameManager"
2. Добавьте скрипт `GameManager`
3. Настройте время игры и другие параметры

### Шаг 7: Добавление финиша
1. Создайте куб или другую зону финиша
2. Добавьте скрипт `FinishZone`
3. Настройте бонусы за завершение
